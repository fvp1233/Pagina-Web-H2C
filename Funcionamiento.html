Inicio Funcionamiento

Funcionamiento del programa

Nuestro programa ha sido creado para facilitar las tareas administrativas en la organización. Con una interfaz simple e intuitiva, permite gestionar recursos, inventarios y tareas de manera eficiente. Además, automatiza procesos clave para ahorrar tiempo y esfuerzo, asegurando una administración fluida y eficaz.

Aspectos generales

En base a las necesidades de las micro-empresas nuestro sistema mantiene diferentes secciones para la automatización de procesos administrativos, principalmente para llevar el control de los movimientos financieros de la empresa.

Al iniciar se le pedira registrar su empresa con datos personales y posteriormente el usuario inicial, el programa mostrará un formulario de login con sus respectivos métodos de recuperación de contraseñas, una vez ingresadas las credenciales mostrara un dashboard como página de inicio y es aqui donde tendremos los diferentes apartados:

  • Planillas
  • Facturas
  • Empleados
  • Clientes
  • Servicios

Vamos a desglozar parte por parte el sistema para entender de mejor manera que hace cada parte de nuestro sistema, presentando la información general del funcionamiento y apartados de cada sección.

Planillas

una planilla es una herramienta que ayuda a organizar y gestionar datos de manera estructurada, facilitando su análisis y uso en diferentes procesos administrativos.

Nuestro programa esta diseñado para poder generar todos los registros de los empleados que existen en la empresa, bien al dar en en generar planilla se llenara automaticamente el datagridview con la informacion del empleado importante recalcar que estas planillas se generan en orden es decir desde la mas antigua a la más nueva, también el registro de un empleado se genera desde el primer mes que fue contratado. Al tener muchos empleados puede ser abrumador esta cantidad de información para esto tenemos la filtración por trimestre, por mes o por estado de la planilla.
El apartado de las planillas tiene las funciones de:

  • Generar planillas
  • Indemnización
  • Actualizar losd datos mostrados del datagridview
  • Eliminar planillas
  • Filtrar una planilla

Todo esto sera mostrado en formato de tabla a través de un datagridview el cual captura la información de cada factura existente.

Facturas

En esta sección se verán todo en relación a facturas, al momento de tener un sistema de este tipo es importante llevar un registro económico; Este apartado permite las funciones de añadir una factura, actualizar la factua y anular una factura. Al querer añadir una nueva factura tendremos que tener mucha información en cuenta.

Mantieniendo las características que pide cada factura se pueden filtrar, por medio del método de pago y del etado en que se encuentren, todo para llevar un control financiero eficiente, facturas tiene que ir asociado a un cliente y un servicio que veremos más a continuación.
El apartado de facturas posee las funciones de:

  • Añadir una Factura
  • Actualizar uan Factura
  • Anular una factura
  • Filtrar una factura

Todo esto sera mostrado en formato de tabla a través de un datagridview el cual captura la información de cada factura existente.

Empleados

En la seccion de usuarios es en donde se maneja todo el control de un usario, aunque su principal función es añadir empleados que se van a ir contratando mediante el tiempo, para esto el apartado permite añadir un nuevo empleado, actualizar un nuevo empleado y eliminar un nuevo empleado.

Existen diferentes filtraciones en esta área como lo son el tipo de empleado, el departamento al que pertence y el estado del empleado, todo esto con el fin de separar entre distinciones a los empleados es decir, cada sección posee sus características definidas
El apartado de usuarios contiene las funciones de:

  • Añadir un empleado
  • Actualizar un empleado
  • Eliminar un empleado
  • Filtrar un empleado

Todo esto sera mostrado en formato de tabla a través de un datagridview el cual captura la información de cada cliente existente.

Clientes

En esta seccón podrás añadir a todos los clientes que tengas, el apartado de clientes permite añadir un cliente con toda su información necesaria, en caso se necesite se puede actualizar dicha información y se puede eliminar un cliente en caso sea deseado.

Para poder actualizar u eliminar un cliente se debe de dar click derecho la fila deseada, aquí se mostrarán las opciones dichas anteriormente, además este apartado cuenta con una filtración de clientes mediante su tipo, o pueden ser buscados directamente en la barra de busqueda.
Esta sección cuenta con las funciones de:

  • Añadir un cliente
  • Actualizar un cliente
  • Eliminar un cliente
  • Filtrar un cliente

Todo esto sera mostrado en formato de tabla a través de un datagridview el cual captura la información de cada cliente existente.

Servicios

El apartado de servicios es un poco más sencillo que las demas secciones ya que tiene pocos campos por llenar;Podemos añadir, eliminar y actualizar un servicio, tal como lo necesitemos

Se nos pedirá el nombre del servicio, su descripción y su precio para poder añadirlo, igualmente el actualizar solo posee estos campos aunque también posee un tipo de servicio, para poder eliminar o actualizar un servicio debemos de dar click derecho en la fila deseada, mostrando estas opciones. También cuenta con filtrado a través del tipo de servicio o si lo desea buscar directamente en la barra de busqueda.
El apartado de clientes cuenta con:

  • Añadir un servicio
  • Actualizar un servicio
  • Eliminar un servicio
  • Filtrar un servicio

Todo esto sera mostrado en formato de tabla a través de un datagridview el cual captura la información de cada servicio existente.
